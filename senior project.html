Assetto Corsa Vehicle Modifier

For my senior design project at California Baptist University, my group was tasked with creating a better way for our SAE Formula team to add cars to their simulator. To solve this problem, we created a simple GUI with python. This GUI allows for the users to see all of the variable modifications in one place instead of going through the cars code line by line. Once the user has made the desired modifications, they are able to save the car and test it out in the game. This will not only make it easier for CBU's SAE Formula Team to add each years build to their simulator, but they can also test out different setups in the simulator before building them.
