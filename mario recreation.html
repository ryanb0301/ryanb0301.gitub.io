NES Mario Recreation

In order to familiarize myself with the basics of video game development I recreated the first level in the classic NES Mario Brothers. I used Unity to implement and run the game. Inkscape was used to create the sprites. This project was helpful in seeing how exactly simple physics could be worked with in Unity.
